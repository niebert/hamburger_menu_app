Icons in Menu

Info Page

Icons can be use in Web Apps in different ways.
  • Font Awesome: the fonts are stored in different font file and the fonts are loaded in general from Content Delivery Networks (CDN). For testing purpose Font Awesome 4.7.0 was used for embedding of icons in offline mode with remote code injection. An AppLSAC uses just the browser as run time environment to execute the locally stored Open Source WebApp. An AppLSAC is designed for offline use without injection of Javascript code from remote servers (e.g. from a CDN).
  • JQuery Mobile Icons: If icons are deployed as pure images they can be loaded and displayed offline. The currently used icons are provided by JQuery Mobile 1.4.5 in the downloaded ZIP file. The JQuery-Mobile folder images/ contains icons in PNG and SVG format. The Scalable Vector Graphics (SVG) are used as image file formats for the icons in this demo.
